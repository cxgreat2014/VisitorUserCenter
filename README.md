# VisitorUserCenter

一个正在实验的项目

##命名缘由:

最初是想命名为UserCenter或者VisitorCenter的,但是因为涉及到短码规范(单词首字母)若是使用UC或者VC
都会造成php解释器解释问题,而且本插件不单单面向用户或者访客,且UC已经和好几款软件重名了,因此最终
选择了一个十分Chinglish的English-like词汇VisitorUserCenter(VUC)
